### **5\. Comparativo das diferenças que você tem do windows:**

| Principais pontos | Zorin OS | Windows |
| :---: | ----- | ----- |
| ***Segurança nativa*** |  *Naturalmente mais seguro contra vírus comuns (porque a estrutura de permissões do usuário é mais rigorosa)* | *Precisa de um sistema de vírus mais potente*  |
| ***Instalações de programas*** | São centralizadas por meio de loja de softwares segura | *São utilizados instaladores da internet* |
| ***Sistemas de atualizações*** | O usuário tem mais controle sobre o que e quando atualizar | *Costumam ser imediatas e muitas vezes obrigatórias* |
| ***Tipo de licença*** | Open source ou código aberto, onde a pessoa pode mudar ou estudar ele | *Proprietária, onde a pessoa pode utilizar o sistema, mas o código continua sendo só da empresa* |
| ***Custos*** | É gratuito na versão Core | É pago, mas o valor normalmente já vem embutido no preço pago pelo computador |

### **6\. Dizer quais seriam as dificuldades de troca de Sistema Operacional Windows para o escolhido:**

Embora o Zorin OS se destaque justamente por proporcionar facilidade de transição do Windows, sempre existem algumas dificuldades, como:

*  Ausência de um suporte nativo (como os mencionados antes, o Adobe e o Microsoft Office), o que exige que os usuários achem outras alternativas ou camadas de compatibilidade   
* Alguns jogos com anti-cheat não funcionam, pois o software do jogo não confia em sistemas que ele não consegue controlar totalmente como ele consegue no Windows   
* Necessidade de adaptação com a mudança no gerenciamento de arquivos  
* Falta de suporte técnico forte como o do WIndows  
* E pessoalmente, enfrentei dificuldades técnicas durante a instalação, como erros de inicialização e conflitos de driver
